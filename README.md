# Weather App

This is a simple weather app made using Node.js.  
It shows current temperature, min temperature, max temperature and weather icon.

I used OpenWeather API to get weather data.

## What it shows
- City name and country
- Current temperature
- Min and max temperature
- Date and time
- Weather icon (day / night)

## Technologies used
- Node.js
- HTML
- CSS
- JavaScript
- OpenWeather API

## How to run project
1. Open project folder in terminal
2. Run `npm install`
3. Run `node index.js`
4. Open browser and go to


## Note
City is set to Noida.  
You can change city name in API URL.

This project is made for practice and learning Node.js.
